# Kondate Loop

1週間の晩ご飯を入力で決め、食材リストを自動生成するローカルWebアプリです。ブラウザで `index.html` を開いて使います。

## 仕様
- 保存先: `localStorage`
- キー形式: `recipe-week-YYYY-MM-DD`
- レシピDB: `recipe-db-v1`
- 週の開始: 月曜起点
- 献立: 晩ご飯のみ（1食に複数品を登録可能）
- レシピ基準人数: 1人前 or 2人前で登録
- 日ごとの人数: 「今日は何人前」で入力
- 食材リスト: 人数に合わせて自動集計（同じ名前+同じ単位は合算、単位が違えば別行）
- URL: 任意
- 食材入力: 食材名 / 数量 / 単位（g, ml, 個, 本, 片, かけ 等）
- 登録動線: 献立画面で料理名を入力し、候補選択 or 新規登録

## 使い方
1. `index.html` を開く
2. 献立ページがTOPで開く
3. 週の開始日を選択する
4. 各日の料理名を入力して品目を追加（候補を選ぶ）
   - 未登録なら「新規登録」からモーダルで詳細入力
5. 必要に応じてレシピDBで食材やURLを編集
   - 「今日は何人前」で人数を入力
6. 自動保存される

## 操作
- 前週/次週ボタンで週移動
- 今週ボタンで現在の週に戻る
- 週をクリアで当週のデータを削除
- 料理名入力で候補を表示（Enterで確定 / 品目追加）
- 食材リストはチェックのみ（人数に合わせて自動生成）
- レシピDBでは登録・編集・削除が可能

## 開発メモ
- HTML/CSS/JSのみで構成
- 追加依存なし
